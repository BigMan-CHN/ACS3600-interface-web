版本：Ver1.4.5
---------------------------------------------------------
时间：2019-12-26 11:32:41
---------------------------------------------------------
问题：
    1.修复打开直播流、关闭直播流、开始录像、停止录像bug
                由于获取通道状态错位，导致控制与当前状态不匹配。
                e.g.控制1通道获取的是2通道状态。
        2.修复打开直播流、关闭直播流、开始录像、停止录像bug
                由于获取通道keyid错位，导致控制与当前状态不匹配。
                e.g.控制1通道获取的是2通道状态。
        3.修复直接开始录制bug，需要在Prepare时传递keyid
        4.添加reboot接口，用于直接system('/sbin/reboot')重启主机
        5.修复在录制中状态下，直接初始化视频流报错问题。
                由于stop时未close，导致直接初始化视频流报错。
---------------------------------------------------------
